# superstore-sales-analysis
End-to-end data analytics project analyzing retail sales data using Python (Pandas, Matplotlib, Seaborn) and Power BI. Includes data cleaning, exploratory data analysis, KPI metrics, and interactive dashboard.

## Objectives
- Analyze sales performance over time
- Identify top-performing regions and categories
- Analyze customer contribution to revenue
- Evaluate shipping performance
- Generate KPIs for business insights
- Build Power BI dashboard

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
- Power BI

## Analysis Performed
- Data Cleaning
- Feature Engineering
- KPI Metrics
- Sales by Region
- Sales by Category
- Monthly Sales Trend
- Shipping Analysis
- Customer Analysis
- Top Products Analysis

## Key Insights
- West region generated the highest sales
- Technology category has the highest revenue
- Top customers contribute a significant portion of total revenue
- Sales peak during end-of-year months
- Standard shipping is most used but slower

## Outputs
The output folder contains:
- Monthly sales trend charts
- Sales by region charts
- Top sub-category charts
- KPI summary
- Aggregated sales tables
